# Airymax Publications — 研究成果发布中心

> **路径**: `docs/Publications/` | **版本**: 0.1.1 | **最后更新**: 2026-07-09

## 概述

`Publications/` 是 Airymax 平台的研究成果发布中心，集中管理 AirymaxRT 和 AirymaxOS 两个项目的所有学术产出，包括期刊论文、会议论文、技术报告和预印本。

## 目录结构

```
docs/Publications/
├── README.md                          # 本文件（研究成果发布中心入口）
├── Journals/                          # 期刊论文
│   ├── README.md                      # 期刊论文清单
│   └── [年份]/[期刊名]/[论文名].md
├── Conferences/                       # 会议论文
│   ├── README.md                      # 会议论文清单
│   └── [年份]/[会议名]/[论文名].md
├── Technical_Reports/                 # 技术报告
│   ├── README.md                      # 技术报告清单
│   └── [年份]/[报告名].md
└── Preprints/                         # 预印本
    ├── README.md                      # 预印本清单
    └── [年份]/[arXiv ID]/README.md    # arXiv 链接及摘要
```

## 分类说明

### 期刊论文（Journals）
经过同行评审的学术期刊论文，包括：
- 发表在 SCI/SSCI 索引期刊上的论文
- 发表在顶级 AI/系统领域期刊上的论文
- 综述文章和特刊论文

### 会议论文（Conferences）
经过同行评审的会议论文，包括：
- 发表在顶级 AI 会议（NeurIPS, ICML, ICLR 等）的论文
- 发表在顶级系统会议（OSDI, SOSP, EuroSys 等）的论文
- 发表在领域特定会议的论文

### 技术报告（Technical Reports）
未经同行评审的技术报告，包括：
- 系统设计与实现报告
- 性能评估报告
- 原型验证报告
- 白皮书

### 预印本（Preprints）
在 arXiv 等预印本平台发布的论文，包括：
- 尚未发表的研究成果
- 已发表论文的预印版本
- 工作进展报告

## 引用规范

引用 Airymax 相关论文时，请遵循以下格式：

### BibTeX 格式
```bibtex
@article{airymax-2026-mcis,
  title={MCIS: Multi-body Control Intelligence System for AI Agent Runtime},
  author={SPHARX Team},
  journal={ACM Transactions on Intelligent Systems and Technology},
  volume={17},
  number={4},
  pages={1--25},
  year={2026},
  publisher={ACM}
}
```

### APA 格式
```
SPHARX Team. (2026). MCIS: Multi-body Control Intelligence System for AI Agent Runtime. 
ACM Transactions on Intelligent Systems and Technology, 17(4), 1-25.
```
